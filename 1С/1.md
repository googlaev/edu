# Практическая работа: Оператор условия в 1С:Предприятие 8

Ниже представлены два варианта заданий для проверки знаний операторов `Если`, `Иначе`, `ИначеЕсли`. Каждый вариант содержит 4 задачи разного типа: поиск ошибок, трассировка кода и написание программы.

---

## Вариант 1

### Задача 1. Найди ошибки
В приведенном ниже коде есть **две** синтаксические ошибки, связанные с оператором условия. Найди их и исправь.

```bsl
Число = 100;
Если Число > 50  // Ошибка 1
    Сообщить("Число большое");
// Ошибка 2
```

### Задача 2. Что выведет программа?
Прочитай код и запиши результат, который появится в окне сообщений.

```bsl
А = 10;
Б = 20;
Если А > Б Тогда
    Сообщить("А больше");
Иначе
    Сообщить("Б больше или равно");
КонецЕсли;
```

### Задача 3. Что выведет программа?
Обрати внимание на цепочку условий `ИначеЕсли`.

```bsl
Оценка = 4;
Если Оценка = 5 Тогда
    Сообщить("Отлично");
ИначеЕсли Оценка = 4 Тогда
    Сообщить("Хорошо");
ИначеЕсли Оценка = 3 Тогда
    Сообщить("Удовлетворительно");
Иначе
    Сообщить("Плохо");
КонецЕсли;
```

### Задача 4. Напиши программу
Напиши код, который проверяет число `X`. 
- Если `X` больше 0, вывести сообщение "Положительное".
- Если `X` меньше 0, вывести сообщение "Отрицательное".
- Если `X` равно 0, вывести сообщение "Ноль".
*(Используй переменную `X = -5` для проверки)*.

## Вариант 2

### Задача 1. Найди ошибки
В приведенном ниже коде есть **две** ошибки в структуре вложенных условий. Найди их и исправь.

```bsl
Логин = "Admin";
Пароль = "123";
Если Логин = "Admin" Тогда
    Если Пароль = "123" Тогда
        Сообщить("Вход разрешен");
    // Ошибка 1
Иначе
    Сообщить("Неверный логин");
КонецЕсли; // Ошибка 2 (лишний или не там стоит)
```

### Задача 2. Что выведет программа?
Обрати внимание на логические операторы `И` и `ИЛИ`.

```bsl
A = 5;
B = 10;
C = 15;
Если A < 10 И B < 10 Тогда
    Сообщить("Условие 1");
Иначе
    Сообщить("Условие 2");
КонецЕсли;
```

### Задача 3. Что выведет программа?
Проанализируй изменение переменной внутри условия.

```bsl
Сумма = 1000;
Если Сумма > 500 Тогда
    Сумма = Сумма + 500;
КонецЕсли;

Если Сумма > 1000 Тогда
    Сообщить("Большая сумма");
Иначе
    Сообщить("Маленькая сумма");
КонецЕсли;
```

### Задача 4. Напиши программу
Напиши код для расчета скидки. 
- Есть переменная `СуммаПокупки`.
- Если `СуммаПокупки` больше 5000, скидка 10%.
- Иначе скидка 0%.
- Выведи итоговую сумму к оплате.
*(Используй переменную `СуммаПокупки = 6000` для проверки)*.
