# Лекция: Работа с массивами в 1С:Предприятие 8

## Введение

**Массив** в 1С — это универсальная коллекция значений. Это один из самых часто используемых типов данных при программировании. 

**Ключевые особенности:**
1.  **Индексация начинается с 0.** Первый элемент имеет индекс `0`, второй — `1` и т.д.
2.  **Динамический размер.** Массив может расти и уменьшаться в процессе выполнения программы.
3.  **Разнородность.** В один массив можно поместить числа, строки, даты, структуры и даже другие массивы (хотя лучше придерживаться одного типа для простоты).

## Часть 1. Работа с числами

Представим, что нам нужно хранить список оценок студентов или цен товаров.

### 1. Создание и добавление элементов

Для создания массива используется конструктор `Новый Массив()`. 
Для добавления элементов есть два основных метода:
*   `.Добавить(Значение)` — добавляет элемент в конец массива.
*   `.Вставить(Индекс, Значение)` — вставляет элемент по конкретному индексу (сдвигая остальные).

```bsl
Процедура Пример_Числа_СозданиеИДобавление()
    // 1. Создание пустого массива
    МассивОценок = Новый Массив;
    
    // 2. Добавление элементов в конец
    МассивОценок.Добавить(5);
    МассивОценок.Добавить(4);
    МассивОценок.Добавить(3);
    
    // 3. Вставка элемента по индексу (например, в начало, под индекс 0)
    // Старый элемент под индексом 0 сдвинется на индекс 1
    МассивОценок.Вставить(0, 2); 
    
    // Проверка количества элементов
    Сообщить("Количество элементов: " + МассивОценок.Количество()); 
    // Ожидается: 4
КонецПроцедуры
```

### 2. Чтение элементов и циклы

Доступ к элементу осуществляется по индексу в квадратных скобках `Массив[Индекс]`.
Для перебора удобнее всего использовать цикл `Для каждого`.

```bsl
Процедура Пример_Числа_ЧтениеИЦиклы()
    МассивЧисел = Новый Массив;
    МассивЧисел.Добавить(10);
    МассивЧисел.Добавить(20);
    МассивЧисел.Добавить(30);
    
    // Способ 1: Классический цикл (если нужен индекс)
    Для Индекс = 0 По МассивЧисел.Количество() - 1 Цикл
        Значение = МассивЧисел[Индекс];
        Сообщить("Индекс: " + Индекс + " | Значение: " + Значение);
    КонецЦикла;
    
    Сообщить("----------");
    
    // Способ 2: Цикл "Для каждого" (если индекс не важен)
    Для каждого Число Из МассивЧисел Цикл
        Сообщить("Значение: " + Число);
    КонецЦикла;
КонецПроцедуры
```

### 3. Удаление элементов

*   `.Удалить(Индекс)` — удаляет элемент по номеру. Элементы после него сдвигаются влево.
*   `.Очистить()` — удаляет все элементы массива.

```bsl
Процедура Пример_Числа_Удаление()
    МассивЧисел = Новый Массив;
    МассивЧисел.Добавить(100);
    МассивЧисел.Добавить(200);
    МассивЧисел.Добавить(300);
    
    // Удаляем элемент под индексом 1 (это число 200)
    МассивЧисел.Удалить(1);
    
    // Теперь в массиве [100, 300]
    Сообщить("После удаления: " + МассивЧисел[0]); // 100
    Сообщить("После удаления: " + МассивЧисел[1]); // 300
    
    // Полная очистка
    МассивЧисел.Очистить();
    Сообщить("Количество после очистки: " + МассивЧисел.Количество()); // 0
КонецПроцедуры
```

### 4. Поиск элемента

Метод `.Найти(Значение)` возвращает **значение** элемента, если он найден, или `Неопределено`, если не найден. 
**Важно:** Он не возвращает индекс! Чтобы найти индекс, нужно использовать цикл.

```bsl
Процедура Пример_Числа_Поиск()
    МассивЧисел = Новый Массив;
    МассивЧисел.Добавить(5);
    МассивЧисел.Добавить(10);
    МассивЧисел.Добавить(15);
    
    // Поиск значения
    НайденноеЗначение = МассивЧисел.Найти(10);
    
    Если НайденноеЗначение <> Неопределено Тогда
        Сообщить("Значение найдено: " + НайденноеЗначение);
    Иначе
        Сообщить("Значение не найдено");
    КонецЕсли;
    
    // Поиск индекса (вручную)
    ИндексПоиска = -1;
    Для Индекс = 0 По МассивЧисел.Количество() - 1 Цикл
        Если МассивЧисел[Индекс] = 15 Тогда
            ИндексПоиска = Индекс;
            Прервать;
        КонецЕсли;
    КонецЦикла;
    
    Сообщить("Индекс числа 15: " + ИндексПоиска);
КонецПроцедуры
```

## Часть 2. Работа со строками

Логика работы та же, но есть нюансы с сравнением строк (регистр) и частыми операциями (конкатенация).

### 1. Создание и заполнение строками

```bsl
Процедура Пример_Строки_Создание()
    СписокИмен = Новый Массив;
    
    // Добавляем строки
    СписокИмен.Добавить("Алексей");
    СписокИмен.Добавить("Мария");
    СписокИмен.Добавить("Дмитрий");
    
    // Доступ по индексу
    ПервоеИмя = СписокИмен[0]; 
    Сообщить("Первое имя: " + ПервоеИмя);
КонецПроцедуры
```

### 2. Поиск и фильтрация строк

При поиске строк важно помнить о регистре. "алексей" не равно "Алексей". Для поиска часто используют функцию `Найти()` внутри цикла.

```bsl
Процедура Пример_Строки_ПоискИФильтрация()
    СписокТоваров = Новый Массив;
    СписокТоваров.Добавить("Ноутбук");
    СписокТоваров.Добавить("Мышка");
    СписокТоваров.Добавить("Клавиатура");
    СписокТоваров.Добавить("Монитор");
    
    // Задача: Найти все товары, содержащие букву "а"
    Сообщить("Товары с буквой 'а':");
    
    Для каждого Товар Из СписокТоваров Цикл
        // Функция Найти возвращает позицию (>0) или 0
        // Приводим к нижнему регистру для надежности поиска
        Если Найти(СтрНиж(Товар), "а") > 0 Тогда
            Сообщить("- " + Товар);
        КонецЕсли;
    КонецЦикла;
КонецПроцедуры
```

### 3. Удаление строк по условию

Частая задача: удалить из массива все элементы, которые не подходят под условие. 
**Важно:** При удалении элементов из массива в цикле нужно быть осторожным с индексами (сдвиг массива). Лучше идти с конца или использовать новый массив.

```bsl
Процедура Пример_Строки_УдалениеПоУсловию()
    СписокИмен = Новый Массив;
    СписокИмен.Добавить("Иван");
    СписокИмен.Добавить("Анна");
    СписокИмен.Добавить("Петр");
    СписокИмен.Добавить("Алина");
    
    // Задача: Оставить только имена, начинающиеся на "А"
    // Способ безопасный: создаем новый массив
    НовыйСписок = Новый Массив;
    
    Для каждого Имя Из СписокИмен Цикл
        Если Лев(Имя, 1) = "А" Тогда
            НовыйСписок.Добавить(Имя);
        КонецЕсли;
    КонецЦикла;
    
    // Заменяем старый массив новым
    СписокИмен = НовыйСписок;
    
    Сообщить("Осталось имен: " + СписокИмен.Количество());
    Для каждого Имя Из СписокИмен Цикл
        Сообщить(Имя);
    КонецЦикла;
КонецПроцедуры
```

## Часть 3. Дополнительные полезные методы

Помимо базовых, есть методы, которые упрощают жизнь программисту 1С.

### 1. Безопасное получение элемента (`.Получить()`)

Если обратиться к `Массив[10]`, а там всего 5 элементов, 1С выдаст ошибку. Метод `.Получить()` вернет `Неопределено` вместо ошибки.

```bsl
Процедура Пример_БезопасныйДоступ()
    Массив = Новый Массив;
    Массив.Добавить(1);
    
    // Это вызовет ошибку, если раскомментировать:
    // Знач = Массив[5]; 
    
    // Это безопасно:
    Знач = Массив.Получить(5);
    Если Знач = Неопределено Тогда
        Сообщить("Элемента с индексом 5 нет");
    КонецЕсли;
КонецПроцедуры
```

### 2. Обмен элементов (`.Обменять()`)

Полезно для сортировки или перемещения элементов.

```bsl
Процедура Пример_Обмен()
    Массив = Новый Массив;
    Массив.Добавить("Первый");
    Массив.Добавить("Второй");
    
    // Меняем местами индекс 0 и 1
    Массив.Обменять(0, 1);
    
    Сообщить(Массив[0]); // Теперь "Второй"
    Сообщить(Массив[1]); // Теперь "Первый"
КонецПроцедуры
```

### 3. Сортировка (`.Сортировать()`)

Массив можно отсортировать. Числа по возрастанию, строки по алфавиту.

```bsl
Процедура Пример_Сортировка()
    МассивЧисел = Новый Массив;
    МассивЧисел.Добавить(50);
    МассивЧисел.Добавить(10);
    МассивЧисел.Добавить(30);
    
    МассивЧисел.Сортировать();
    
    Для каждого Ч Из МассивЧисел Цикл
        Сообщить(Ч); // 10, 30, 50
    КонецЦикла;
КонецПроцедуры
```

## Методы

| Действие | Метод / Синтаксис | Примечание |
| :--- | :--- | :--- |
| **Создание** | `Новый Массив()` | |
| **Добавить** | `.Добавить(Знач)` | В конец |
| **Вставить** | `.Вставить(Индекс, Знач)` | Со сдвигом |
| **Получить** | `Массив[Индекс]` | Ошибка если нет индекса |
| **Получить (без ошибки)** | `.Получить(Индекс)` | Вернет `Неопределено` |
| **Количество** | `.Количество()` | Размер массива |
| **Удалить** | `.Удалить(Индекс)` | Сдвигает остальные |
| **Очистить** | `.Очистить()` | Удаляет все |
| **Найти значение** | `.Найти(Знач)` | Возвращает значение или `Неопределено` |
| **Обменять** | `.Обменять(Ин1, Ин2)` | Меняет местами |
| **Сортировать** | `.Сортировать()` | По возрастанию |
