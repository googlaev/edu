## 1. Введение в многопоточность

**Поток (thread)** — это наименьшая единица исполнения внутри процесса. В отличие от процессов, потоки одного процесса разделяют общую память: глобальные переменные, кучу (heap), файловые дескрипторы. У каждого потока собственный стек и регистры процессора.

### Зачем нужна многопоточность?

| Причина | Описание |
|---|---|
| **Параллелизм** | Выполнение нескольких задач одновременно на нескольких ядрах |
| **Отзывчивость** | UI-поток не блокируется фоновыми вычислениями |
| **Разделение задач** | Чтение данных, обработка, вывод — в разных потоках |
| **Производительность** | Ускорение CPU-bound и I/O-bound задач |

### Процесс vs Поток


Процесс A                          Процесс B
┌──────────────────────────┐       ┌──────────────┐
│  Код | Данные | Куча     │       │  ...         │
│  ─────────────────────   │       └──────────────┘
│  Поток 1 │ Поток 2       │
│  стек 1  │ стек 2        │
└──────────────────────────┘


## 2. Основы pthreads

В стандарте POSIX многопоточность реализована через библиотеку **pthreads** (`<pthread.h>`).

### Компиляция

```bash
gcc -o program program.c -lpthread
```

Флаг `-lpthread` обязателен — он подключает библиотеку потоков.

### Основные типы

```c
pthread_t          // Идентификатор потока
pthread_mutex_t    // Мьютекс
pthread_cond_t     // Условная переменная
pthread_barrier_t  // Барьер
pthread_attr_t     // Атрибуты потока
```

## 3. Создание и завершение потоков

### 3.1 Простейший пример — Hello from threads

```c
#include <stdio.h>
#include <pthread.h>

// Функция, которую будет выполнять поток
// Принимает void* и возвращает void*
void *hello(void *arg) {
    int id = *(int *)arg;
    printf("Привет из потока %d!\n", id);
    return NULL;
}

int main(void) {
    pthread_t t1, t2;
    int id1 = 1, id2 = 2;

    // pthread_create(поток, атрибуты, функция, аргумент)
    pthread_create(&t1, NULL, hello, &id1);
    pthread_create(&t2, NULL, hello, &id2);

    // Ждём завершения потоков
    pthread_join(t1, NULL);
    pthread_join(t2, NULL);

    printf("Все потоки завершены.\n");
    return 0;
}
```

**Вывод** (порядок может меняться):
```
Привет из потока 1!
Привет из потока 2!
Все потоки завершены.
```

### 3.2 Передача аргументов и получение результата

```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>

typedef struct {
    int   start;
    int   end;
    long  result; // сюда поток запишет результат
} SumArgs;

// Считаем сумму чисел от start до end
void *sum_range(void *arg) {
    SumArgs *a = (SumArgs *)arg;
    a->result = 0;
    for (int i = a->start; i <= a->end; i++) {
        a->result += i;
    }
    return NULL; // можно вернуть &a->result, но структура удобнее
}

int main(void) {
    pthread_t t1, t2;

    SumArgs a1 = {1,   5000, 0};
    SumArgs a2 = {5001, 10000, 0};

    pthread_create(&t1, NULL, sum_range, &a1);
    pthread_create(&t2, NULL, sum_range, &a2);

    pthread_join(t1, NULL);
    pthread_join(t2, NULL);

    printf("Сумма 1..10000 = %ld\n", a1.result + a2.result);
    // Ожидаемый результат: 50005000
    return 0;
}
```

### 3.3 Создание N потоков в цикле

```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>

#define NUM_THREADS 5

void *worker(void *arg) {
    int id = (int)(size_t)arg; // передаём int через указатель
    printf("Поток %d работает\n", id);
    return NULL;
}

int main(void) {
    pthread_t threads[NUM_THREADS];

    for (int i = 0; i < NUM_THREADS; i++) {
        // Внимание: передаём i через (void*)(size_t)i,
        // чтобы избежать проблемы с захватом адреса переменной цикла
        pthread_create(&threads[i], NULL, worker, (void *)(size_t)i);
    }

    for (int i = 0; i < NUM_THREADS; i++) {
        pthread_join(threads[i], NULL);
    }

    return 0;
}
```

> ⚠️ **Антипаттерн**: передавать `&i` из цикла — опасно! К моменту запуска потока `i` уже может смениться. Всегда передавайте значение через `(void*)(size_t)i` или копируйте в отдельную переменную.

## 4. Мьютексы — защита общих данных

**Состояние гонки (race condition)** возникает, когда два потока одновременно читают и изменяют одну переменную. Результат непредсказуем.

### 4.1 Пример гонки (неправильно!)

```c
#include <stdio.h>
#include <pthread.h>

long counter = 0; // общая переменная

void *increment(void *arg) {
    for (int i = 0; i < 1000000; i++) {
        counter++; // НЕ атомарная операция! READ → MODIFY → WRITE
    }
    return NULL;
}

int main(void) {
    pthread_t t1, t2;
    pthread_create(&t1, NULL, increment, NULL);
    pthread_create(&t2, NULL, increment, NULL);
    pthread_join(t1, NULL);
    pthread_join(t2, NULL);
    // Ожидаем 2000000, но получим меньше из-за гонки!
    printf("counter = %ld\n", counter);
    return 0;
}
```

### 4.2 Исправление с мьютексом

```c
#include <stdio.h>
#include <pthread.h>

long counter = 0;
pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER; // статическая инициализация

void *increment(void *arg) {
    for (int i = 0; i < 1000000; i++) {
        pthread_mutex_lock(&mutex);   // захватываем мьютекс
        counter++;                    // критическая секция
        pthread_mutex_unlock(&mutex); // освобождаем мьютекс
    }
    return NULL;
}

int main(void) {
    pthread_t t1, t2;
    pthread_create(&t1, NULL, increment, NULL);
    pthread_create(&t2, NULL, increment, NULL);
    pthread_join(t1, NULL);
    pthread_join(t2, NULL);
    printf("counter = %ld\n", counter); // Всегда 2000000
    pthread_mutex_destroy(&mutex);
    return 0;
}
```

### 4.4 Несколько мьютексов — защита разных ресурсов

```c
#include <stdio.h>
#include <pthread.h>
#include <unistd.h>

typedef struct {
    double  balance;
    pthread_mutex_t lock;
} Account;

void transfer(Account *from, Account *to, double amount) {
    // Всегда захватываем мьютексы в одном порядке (по адресу),
    // чтобы избежать deadlock!
    Account *first  = (from < to) ? from : to;
    Account *second = (from < to) ? to   : from;

    pthread_mutex_lock(&first->lock);
    pthread_mutex_lock(&second->lock);

    if (from->balance >= amount) {
        from->balance -= amount;
        to->balance   += amount;
        printf("Перевод %.2f: OK\n", amount);
    } else {
        printf("Недостаточно средств\n");
    }

    pthread_mutex_unlock(&second->lock);
    pthread_mutex_unlock(&first->lock);
}

int main(void) {
    Account alice = {1000.0, PTHREAD_MUTEX_INITIALIZER};
    Account bob   = {500.0,  PTHREAD_MUTEX_INITIALIZER};

    transfer(&alice, &bob, 200.0);
    printf("Alice: %.2f, Bob: %.2f\n", alice.balance, bob.balance);

    pthread_mutex_destroy(&alice.lock);
    pthread_mutex_destroy(&bob.lock);
    return 0;
}
```

## 5. Условные переменные

**Условная переменная** позволяет потоку ждать наступления какого-либо условия. Используется всегда вместе с мьютексом.

### 5.1 Паттерн «производитель — потребитель»

```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>
#include <unistd.h>

#define BUFFER_SIZE 5

int   buffer[BUFFER_SIZE];
int   count = 0; // количество элементов в буфере

pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;
pthread_cond_t  not_full  = PTHREAD_COND_INITIALIZER; // буфер не полон
pthread_cond_t  not_empty = PTHREAD_COND_INITIALIZER; // буфер не пуст

// Производитель добавляет данные
void *producer(void *arg) {
    for (int i = 1; i <= 10; i++) {
        pthread_mutex_lock(&mutex);

        // Ждём, пока в буфере появится место
        while (count == BUFFER_SIZE) {
            pthread_cond_wait(&not_full, &mutex);
            // pthread_cond_wait атомарно:
            //   1) освобождает mutex
            //   2) усыпляет поток
            //   3) при пробуждении — снова захватывает mutex
        }

        buffer[count++] = i;
        printf("[Producer] добавил %d (в буфере: %d)\n", i, count);

        pthread_cond_signal(&not_empty); // будим потребителя
        pthread_mutex_unlock(&mutex);

        usleep(100000); // 100 мс имитация работы
    }
    return NULL;
}

// Потребитель забирает данные
void *consumer(void *arg) {
    int total = 0;
    for (int i = 0; i < 10; i++) {
        pthread_mutex_lock(&mutex);

        // Ждём, пока в буфере появятся данные
        while (count == 0) {
            pthread_cond_wait(&not_empty, &mutex);
        }

        int val = buffer[--count];
        total += val;
        printf("[Consumer] взял %d (в буфере: %d)\n", val, count);

        pthread_cond_signal(&not_full); // будим производителя
        pthread_mutex_unlock(&mutex);

        usleep(200000); // 200 мс имитация обработки
    }
    printf("[Consumer] Сумма всех значений: %d\n", total);
    return NULL;
}

int main(void) {
    pthread_t prod, cons;
    pthread_create(&prod, NULL, producer, NULL);
    pthread_create(&cons, NULL, consumer, NULL);
    pthread_join(prod, NULL);
    pthread_join(cons, NULL);

    pthread_mutex_destroy(&mutex);
    pthread_cond_destroy(&not_full);
    pthread_cond_destroy(&not_empty);
    return 0;
}
```

### 5.2 Широковещательный сигнал (broadcast)

```c
#include <stdio.h>
#include <pthread.h>
#include <unistd.h>

#define NUM_WORKERS 4

int ready = 0;
pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;
pthread_cond_t  go    = PTHREAD_COND_INITIALIZER;

void *worker(void *arg) {
    int id = (int)(size_t)arg;
    pthread_mutex_lock(&mutex);
    printf("Рабочий %d ждёт сигнала...\n", id);
    while (!ready) {
        pthread_cond_wait(&go, &mutex); // ждём стартового сигнала
    }
    pthread_mutex_unlock(&mutex);
    printf("Рабочий %d начал работу!\n", id);
    return NULL;
}

int main(void) {
    pthread_t workers[NUM_WORKERS];
    for (int i = 0; i < NUM_WORKERS; i++) {
        pthread_create(&workers[i], NULL, worker, (void *)(size_t)i);
    }

    sleep(1); // ждём, пока все потоки войдут в ожидание

    pthread_mutex_lock(&mutex);
    ready = 1;
    pthread_cond_broadcast(&go); // будим ВСЕХ ожидающих потоков
    pthread_mutex_unlock(&mutex);

    for (int i = 0; i < NUM_WORKERS; i++) {
        pthread_join(workers[i], NULL);
    }
    pthread_mutex_destroy(&mutex);
    pthread_cond_destroy(&go);
    return 0;
}
```

## 6. Семафоры

**Семафор** — счётчик с двумя операциями: `sem_wait` (уменьшить, заблокировать при 0) и `sem_post` (увеличить). Объявлен в `<semaphore.h>`.

```c
#include <stdio.h>
#include <pthread.h>
#include <semaphore.h>
#include <unistd.h>

#define MAX_CONNECTIONS 3   // максимум одновременных "подключений"
#define NUM_CLIENTS     7

sem_t semaphore;

void *client(void *arg) {
    int id = (int)(size_t)arg;
    printf("Клиент %d ожидает подключения...\n", id);

    sem_wait(&semaphore); // захватываем слот (если 0 — блокируемся)

    printf("Клиент %d подключён!\n", id);
    sleep(1); // имитация работы
    printf("Клиент %d отключился\n", id);

    sem_post(&semaphore); // освобождаем слот
    return NULL;
}

int main(void) {
    sem_init(&semaphore, 0, MAX_CONNECTIONS); // начальное значение = 3

    pthread_t clients[NUM_CLIENTS];
    for (int i = 0; i < NUM_CLIENTS; i++) {
        pthread_create(&clients[i], NULL, client, (void *)(size_t)i);
        usleep(50000);
    }

    for (int i = 0; i < NUM_CLIENTS; i++) {
        pthread_join(clients[i], NULL);
    }

    sem_destroy(&semaphore);
    return 0;
}
```

### Семафор как сигнал между потоками

```c
#include <stdio.h>
#include <pthread.h>
#include <semaphore.h>

sem_t sem;

void *waiter(void *arg) {
    printf("Поток ожидает события...\n");
    sem_wait(&sem);           // блокируется, пока sem == 0
    printf("Событие получено!\n");
    return NULL;
}

void *signaler(void *arg) {
    printf("Поток отправляет событие\n");
    sem_post(&sem);           // увеличивает sem, будит waiter
    return NULL;
}

int main(void) {
    sem_init(&sem, 0, 0); // начальное значение = 0

    pthread_t t1, t2;
    pthread_create(&t1, NULL, waiter,   NULL);
    pthread_create(&t2, NULL, signaler, NULL);
    pthread_join(t1, NULL);
    pthread_join(t2, NULL);
    sem_destroy(&sem);
    return 0;
}
```

## 7. Барьеры

**Барьер** синхронизирует N потоков в одной точке: ни один не продолжит выполнение, пока все N не достигнут барьера.

```c
#include <stdio.h>
#include <pthread.h>
#include <unistd.h>

#define NUM_THREADS 4

pthread_barrier_t barrier;

void *phase_worker(void *arg) {
    int id = (int)(size_t)arg;

    // --- Фаза 1 ---
    sleep(id % 3); // имитируем разное время выполнения
    printf("Поток %d завершил фазу 1\n", id);

    // Все потоки ждут здесь
    pthread_barrier_wait(&barrier);

    // --- Фаза 2 (начинается только после завершения всеми фазы 1) ---
    printf("Поток %d начал фазу 2\n", id);

    return NULL;
}

int main(void) {
    pthread_barrier_init(&barrier, NULL, NUM_THREADS);

    pthread_t threads[NUM_THREADS];
    for (int i = 0; i < NUM_THREADS; i++) {
        pthread_create(&threads[i], NULL, phase_worker, (void *)(size_t)i);
    }

    for (int i = 0; i < NUM_THREADS; i++) {
        pthread_join(threads[i], NULL);
    }

    pthread_barrier_destroy(&barrier);
    return 0;
}
```

## 8. Задачи и паттерны

### 8.1 Пул потоков (Thread Pool)

Создаём N потоков один раз и раздаём им задачи из очереди — без накладных расходов на создание/уничтожение потоков.

```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>
#include <unistd.h>

#define POOL_SIZE   3
#define QUEUE_SIZE  10

typedef void (*task_func)(void *);

typedef struct {
    task_func func;
    void     *arg;
} Task;

typedef struct {
    Task            queue[QUEUE_SIZE];
    int             head, tail, count;
    pthread_mutex_t lock;
    pthread_cond_t  not_empty;
    pthread_cond_t  not_full;
    int             shutdown; // флаг завершения
} ThreadPool;

ThreadPool pool;

// Рабочий поток: бесконечно берёт задачи из очереди
void *pool_worker(void *arg) {
    while (1) {
        pthread_mutex_lock(&pool.lock);

        while (pool.count == 0 && !pool.shutdown) {
            pthread_cond_wait(&pool.not_empty, &pool.lock);
        }

        if (pool.shutdown && pool.count == 0) {
            pthread_mutex_unlock(&pool.lock);
            break;
        }

        Task t = pool.queue[pool.head];
        pool.head = (pool.head + 1) % QUEUE_SIZE;
        pool.count--;

        pthread_cond_signal(&pool.not_full);
        pthread_mutex_unlock(&pool.lock);

        t.func(t.arg); // выполняем задачу вне мьютекса
    }
    return NULL;
}

// Добавить задачу в очередь
void pool_submit(task_func func, void *arg) {
    pthread_mutex_lock(&pool.lock);
    while (pool.count == QUEUE_SIZE) {
        pthread_cond_wait(&pool.not_full, &pool.lock);
    }
    pool.queue[pool.tail].func = func;
    pool.queue[pool.tail].arg  = arg;
    pool.tail  = (pool.tail + 1) % QUEUE_SIZE;
    pool.count++;
    pthread_cond_signal(&pool.not_empty);
    pthread_mutex_unlock(&pool.lock);
}

// --- Пример задачи ---
void print_task(void *arg) {
    int n = (int)(size_t)arg;
    printf("Задача %d выполнена потоком\n", n);
    usleep(200000);
}

int main(void) {
    // Инициализация
    pool.head = pool.tail = pool.count = pool.shutdown = 0;
    pthread_mutex_init(&pool.lock, NULL);
    pthread_cond_init(&pool.not_empty, NULL);
    pthread_cond_init(&pool.not_full, NULL);

    pthread_t workers[POOL_SIZE];
    for (int i = 0; i < POOL_SIZE; i++) {
        pthread_create(&workers[i], NULL, pool_worker, NULL);
    }

    // Отправляем 8 задач
    for (int i = 0; i < 8; i++) {
        pool_submit(print_task, (void *)(size_t)i);
    }

    // Завершение: дождаться опустошения очереди, затем сигнал shutdown
    sleep(2);
    pthread_mutex_lock(&pool.lock);
    pool.shutdown = 1;
    pthread_cond_broadcast(&pool.not_empty);
    pthread_mutex_unlock(&pool.lock);

    for (int i = 0; i < POOL_SIZE; i++) {
        pthread_join(workers[i], NULL);
    }

    pthread_mutex_destroy(&pool.lock);
    pthread_cond_destroy(&pool.not_empty);
    pthread_cond_destroy(&pool.not_full);
    printf("Пул завершён.\n");
    return 0;
}
```

### 8.2 Параллельное суммирование массива

```c
#include <stdio.h>
#include <stdlib.h>
#include <pthread.h>

#define ARR_SIZE    100000000  // 100 миллионов элементов
#define NUM_THREADS 4

long long array[ARR_SIZE];

typedef struct {
    int       start;
    int       end;
    long long partial_sum;
} SumTask;

void *partial_sum(void *arg) {
    SumTask *t = (SumTask *)arg;
    t->partial_sum = 0;
    for (int i = t->start; i < t->end; i++) {
        t->partial_sum += array[i];
    }
    return NULL;
}

int main(void) {
    // Заполняем массив
    for (int i = 0; i < ARR_SIZE; i++) array[i] = 1;

    pthread_t threads[NUM_THREADS];
    SumTask   tasks[NUM_THREADS];
    int chunk = ARR_SIZE / NUM_THREADS;

    for (int i = 0; i < NUM_THREADS; i++) {
        tasks[i].start = i * chunk;
        tasks[i].end   = (i == NUM_THREADS - 1) ? ARR_SIZE : (i + 1) * chunk;
        pthread_create(&threads[i], NULL, partial_sum, &tasks[i]);
    }

    long long total = 0;
    for (int i = 0; i < NUM_THREADS; i++) {
        pthread_join(threads[i], NULL);
        total += tasks[i].partial_sum;
    }

    printf("Сумма = %lld (ожидается %d)\n", total, ARR_SIZE);
    return 0;
}
```

### 8.3 Читатели–писатели (Read-Write Lock)

Несколько читателей могут работать одновременно, но писатель — эксклюзивно.

```c
#include <stdio.h>
#include <pthread.h>
#include <unistd.h>

pthread_rwlock_t rwlock = PTHREAD_RWLOCK_INITIALIZER;
int shared_data = 0;

void *reader(void *arg) {
    int id = (int)(size_t)arg;
    for (int i = 0; i < 3; i++) {
        pthread_rwlock_rdlock(&rwlock); // несколько читателей — одновременно
        printf("Читатель %d читает: %d\n", id, shared_data);
        usleep(100000);
        pthread_rwlock_unlock(&rwlock);
        usleep(50000);
    }
    return NULL;
}

void *writer(void *arg) {
    int id = (int)(size_t)arg;
    for (int i = 0; i < 2; i++) {
        pthread_rwlock_wrlock(&rwlock); // писатель — эксклюзивно
        shared_data += 10;
        printf("Писатель %d записал: %d\n", id, shared_data);
        usleep(200000);
        pthread_rwlock_unlock(&rwlock);
        usleep(300000);
    }
    return NULL;
}

int main(void) {
    pthread_t r[3], w[2];
    for (int i = 0; i < 3; i++) pthread_create(&r[i], NULL, reader, (void *)(size_t)i);
    for (int i = 0; i < 2; i++) pthread_create(&w[i], NULL, writer, (void *)(size_t)i);

    for (int i = 0; i < 3; i++) pthread_join(r[i], NULL);
    for (int i = 0; i < 2; i++) pthread_join(w[i], NULL);

    pthread_rwlock_destroy(&rwlock);
    return 0;
}
```

## 9. Типичные ошибки

### 9.1 Взаимная блокировка (Deadlock)

```c
// ❌ ОПАСНО: разный порядок захвата мьютексов → deadlock
// Поток 1: lock(A) → lock(B)
// Поток 2: lock(B) → lock(A)  ← оба застрянут!

// ✅ ПРАВИЛЬНО: всегда один порядок захвата
pthread_mutex_lock(&mutex_A);
pthread_mutex_lock(&mutex_B);
// ... работа ...
pthread_mutex_unlock(&mutex_B);
pthread_mutex_unlock(&mutex_A);
```

### 9.2 Ложный сигнал (Spurious Wakeup)

```c
// ❌ НЕПРАВИЛЬНО: if может не уловить повторную проверку
if (count == 0) {
    pthread_cond_wait(&cond, &mutex);
}

// ✅ ПРАВИЛЬНО: всегда while!
while (count == 0) {
    pthread_cond_wait(&cond, &mutex);
}
```

### 9.3 Утечка потока (Thread Leak)

```c
// ❌ НЕПРАВИЛЬНО: поток не присоединён и не отсоединён → утечка ресурсов
pthread_create(&tid, NULL, worker, NULL);
// забыли join или detach!

// ✅ ПРАВИЛЬНО — вариант 1: join
pthread_join(tid, NULL);

// ✅ ПРАВИЛЬНО — вариант 2: detach (для "огонь и забудь")
pthread_detach(tid);
// или через атрибуты при создании:
pthread_attr_t attr;
pthread_attr_init(&attr);
pthread_attr_setdetachstate(&attr, PTHREAD_CREATE_DETACHED);
pthread_create(&tid, &attr, worker, NULL);
pthread_attr_destroy(&attr);
```

### 9.4 Захват адреса переменной цикла

```c
// ❌ НЕПРАВИЛЬНО: все потоки получат одно и то же значение i
for (int i = 0; i < N; i++) {
    pthread_create(&threads[i], NULL, worker, &i); // опасно!
}

// ✅ ПРАВИЛЬНО: передаём значение через указатель
for (int i = 0; i < N; i++) {
    pthread_create(&threads[i], NULL, worker, (void *)(size_t)i);
}
```

### 9.5 Таблица ошибок и решений

| Ошибка | Причина | Решение |
|---|---|---|
| Race condition | Нет синхронизации | Мьютекс / атомики |
| Deadlock | Неправильный порядок мьютексов | Фиксированный порядок захвата |
| Spurious wakeup | `if` вместо `while` | Всегда `while` с `cond_wait` |
| Thread leak | Нет `join`/`detach` | Всегда join или detach |
| Stack overflow | Слишком глубокая рекурсия в потоке | Увеличить стек через `attr` |

## 10. Итоги и рекомендации

### Краткая шпаргалка по функциям

```
Потоки:
  pthread_create()   — создать поток
  pthread_join()     — ждать завершения
  pthread_detach()   — отсоединить (не ждать)
  pthread_exit()     — завершить текущий поток
  pthread_self()     — получить свой pthread_t

Мьютексы:
  pthread_mutex_init()    — инициализация
  pthread_mutex_lock()    — захват (блокирующий)
  pthread_mutex_trylock() — попытка захвата (неблокирующий)
  pthread_mutex_unlock()  — освобождение
  pthread_mutex_destroy() — уничтожение

Условные переменные:
  pthread_cond_init()      — инициализация
  pthread_cond_wait()      — ждать сигнала (освобождает mutex)
  pthread_cond_signal()    — разбудить один поток
  pthread_cond_broadcast() — разбудить все потоки
  pthread_cond_destroy()   — уничтожение

Семафоры:
  sem_init()   — инициализация
  sem_wait()   — P-операция (уменьшить / заблокировать)
  sem_post()   — V-операция (увеличить / разбудить)
  sem_destroy()— уничтожение

Барьеры:
  pthread_barrier_init()  — инициализация (указать N потоков)
  pthread_barrier_wait()  — ждать всех N потоков
  pthread_barrier_destroy()— уничтожение
```

### Правила хорошего многопоточного кода

1. **Минимизируйте критические секции** — держите `lock`/`unlock` как можно ближе друг к другу.
2. **Фиксируйте порядок захвата мьютексов** — всегда один и тот же порядок, чтобы избежать deadlock.
3. **Используйте `while` с `cond_wait`** — никогда `if`.
4. **Всегда делайте `join` или `detach`** — иначе утечка ресурсов.
5. **Не передавайте `&i` из цикла** — передавайте значение через `(void*)(size_t)i`.
6. **Компилируйте с `-pthread`** — не `-lpthread` наедине, а именно флаг компилятора.
7. **Используйте `valgrind --tool=helgrind`** — для поиска гонок и deadlock в отладке.
