## ⭐ 11. Вычисление корней квадратного уравнения

Даны коэффициенты `a`, `b`, `c` (double). Найти корни уравнения:

**ax² + bx + c = 0**

Условия: - Если `a = 0` → ошибка - Если D < 0 → корней нет - Если D = 0
→ один корень - Если D > 0 → два корня

**Пример**

    Ввод:
    1 -3 2
    Вывод:
    x1 = 2
    x2 = 1

------------------------------------------------------------------------

## ⭐ 12. Проверка, попадает ли точка в круг

Ввод: `x`, `y`, `r` (float).
Круг с центром (0,0).
Проверить положение точки.

Формула: `x² + y² <= r²`

**Примеры**

    Ввод: 1 1 2
    Вывод: Точка внутри круга

    Ввод: 3 0 3
    Вывод: Точка на границе

    Ввод: 5 5 3
    Вывод: Точка вне круга

------------------------------------------------------------------------

## ⭐ 15. Калькулятор площадей фигур

Фигура вводится как символ:

-   `c` --- круг, вводится радиус
-   `s` --- квадрат, вводится сторона
-   `t` --- равносторонний треугольник, вводится сторона

Формулы: - Круг: S = π * r²
- Квадрат: S = a²
- Треугольник: S = (sqrt(3)/4) * a²

**Примеры**

    Ввод:
    c
    3
    Вывод: Площадь = 28.2743

    Ввод:
    s
    4
    Вывод: Площадь = 16

    Ввод:
    t
    2
    Вывод: Площадь = 1.732
